Introduction

The following analysis is presented as part of the JHU Reproducible Research course, Peer Assessment 1. The source data represents personal movement activity observations of a single individual, collected in 5-minute intervals daily. The time period for the data collection spans October 2012 through November 2012.

Imputing missing values

Calculate and report the total number of missing values in the dataset

sum(is.na(activity.data$steps))

## [1] 2304

The strategy for filling in all of the missing values in the dataset is to use the mean for a given 5-minute interval.

activity.imputed <- activity.data

for (i in 1:nrow(activity.imputed)) {

    if (is.na(activity.imputed$steps[i])) {
        activity.imputed$steps[i] <- activity.int.steps.mean$steps[
            which(
                activity.imputed $interval[i] == activity.int.steps.mean$interval
            )
        ]
    }
}

Group the observations by date

Are there differences in activity patterns between weekdays and weekends?

Using the previous imputed dataset with the filled-in missing values, create a new factor variable in the dataset with two levels – “weekday” and “weekend” indicating whether a given date is a weekday or weekend day

activity.imputed <- activity.imputed %>% 
    mutate(day_type = factor(ifelse(
